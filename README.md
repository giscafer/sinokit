Vue.js 2.x 组件库、基础组件、业务组件、区块、模板

组件文档：http://blog.giscafer.com/sinokit

## Usage

### demo

```
npm run start
```

### Docs

```
docsify init ./docs
npm run docs
```

### Build

```
npm run build
```
